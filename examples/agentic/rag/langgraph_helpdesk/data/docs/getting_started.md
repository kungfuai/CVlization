# Getting Started with CVlization

Welcome to CVlization! This guide introduces the essential steps for running examples and customizing them for your projects.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/kungfuai/CVlization
   ```
2. Install the CLI (optional):
   ```
   pip install .
   ```

## Running Examples

- Use `cvl list` to browse available recipes.
- Each example contains `build.sh`, `train.sh`, and `predict.sh`.
- Build images with Docker before running training or inference.

## Working with Worktrees

Use `git worktree add <path> main` to checkout additional branches alongside your main repository.
