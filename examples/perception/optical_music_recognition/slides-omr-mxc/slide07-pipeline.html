OpenScore Dataset Pipeline

Source: OpenScore corpus (1,460 lieder, 122 quartets, 94 orchestra scores)

Render: MusicXML → LilyPond → PNG page images (835x1181 px)

Slice: music21 splits MusicXML per page using SVG bar numbers

Result: 14,456 (image, MusicXML) pairs on HuggingFace (zzsi/openscore)

Also: OLiMPiC dataset — 18K system-level piano crops with MusicXML

Pipeline

MusicXML → LilyPond Docker → SVG pages → bar number extraction → per-page MusicXML slice → HuggingFace push
