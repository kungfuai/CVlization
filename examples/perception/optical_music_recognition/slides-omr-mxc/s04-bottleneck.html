The Token Bottleneck

The Problem

  • MusicXML is extremely verbose (~14K tokens per page)
  • VLM context window: 4,096 tokens
  • Only 10% of pages fit — models never reach <note> elements
  • All models generate headers/metadata, never actual notes

The Impact

  • 6 models tested with XML targets
  • 0% pitch accuracy across all models
  • Models learn structure but not musical content
  • Need 12× compression to fit notes in context
